Every command you need

Run these inside Claude Code after bootstrapping the framework.

/prototype "idea" Working demo in hours — login, core feature, seeded data. Validate with real users before committing.
/startup "idea" Full pipeline — market analysis → PRD → design + architecture → engineering team.
/build <project> Kick off the engineering pipeline for an existing project. Vertical slices with quality gates.
/ideate "idea" Strategic phase only — business analysis + PRD. Clear GO/NO-GO before investing further.
/iterate <project> Improve a working product — new features or refinements through the full pipeline.
/review-pr <url> Full 2-agent PR review — security engineer + architecture reviewer run in parallel.
